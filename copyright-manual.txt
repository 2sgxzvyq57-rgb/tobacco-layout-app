烟草许可证实地核查布局图生成软件 V1.0
软件说明书
============================================================

一、软件概述
------------------------------------------------------------
烟草许可证实地核查布局图生成软件是一款基于语音AI的店面布局图快速生成工具，
专为烟草许可证实地核查场景设计。用户通过语音描述店面布局，
AI自动解析并生成按比例绘制的布局图，支持导出PNG图片。

二、功能特点
------------------------------------------------------------
1. 语音输入：支持语音识别，快速输入店面描述
2. AI智能解析：自动提取店面尺寸、门位置、物体布局
3. 布局图生成：按比例精确绘制店面平面图
4. 交互编辑：支持拖拽、旋转、调整物体大小
5. 语音备注：支持语音添加备注信息
6. 楼梯识别：自动识别并显示楼梯位置
7. 图片导出：支持导出高清PNG图片
8. PWA支持：可添加到手机主屏幕，像App一样使用

三、技术架构
------------------------------------------------------------
1. 前端框架：Next.js 16 (App Router)
2. 核心语言：TypeScript 5 + React 19
3. UI组件：Tailwind CSS 4 + shadcn/ui
4. AI服务：豆包大语言模型API
5. 语音识别：Web Speech API
6. 图形绘制：Canvas API 2D

四、使用说明
------------------------------------------------------------
1. 打开应用，点击麦克风按钮开始语音输入
2. 描述店面布局（朝向、尺寸、门位置、物体位置）
3. 点击"生成布局图"，AI解析并绘制
4. 拖拽物体调整位置，使用控制面板调整大小和方向
5. 点击"导出图片"下载PNG

五、运行环境
------------------------------------------------------------
- 操作系统：Windows / macOS / Linux / Android / iOS
- 浏览器：Chrome 90+ / Safari 14+ / Edge 90+
- 网络：需要互联网连接（AI解析需要）
- 存储：本地存储布局数据
